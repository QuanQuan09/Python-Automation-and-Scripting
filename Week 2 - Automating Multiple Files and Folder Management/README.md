# Python Automation Bootcamp - Week 2
## Automating Multiple Files & Folder Management

## Overview

This week focused on automating common file system tasks using Python. I learned how to manage directories, process multiple files automatically, create backups, rename files in batches, and build a complete automation utility that combines these skills.

## Skills Learned

- Using the `os` module for directory management
- Working with file paths using `os.path.join()`
- Batch processing multiple files with `os.listdir()`
- Filtering files by extension
- Reading multiple CSV files using `csv.DictReader()`
- Creating backup folders and copying files with `shutil`
- Renaming files automatically with `os.rename()`
- Generating summary reports
- Creating organized project folder structures
- Debugging file path and automation issues
