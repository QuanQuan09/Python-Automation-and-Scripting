# Week 1 – Python as a Productivity Tool

## Overview

Week 1 introduces the fundamentals of Python automation and establishes the core concepts used throughout the bootcamp.

This week focuses on understanding how Python can automate repetitive tasks by working with files, folders, and simple reporting scripts.

---

## Topics Covered

- Python as a Productivity Tool
- Reading Files
- Writing Files
- Appending Files
- Working with Directories
- Listing Files
- Creating Folders
- File Inventory Automation
- Report Generation
